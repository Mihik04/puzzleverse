# PuzzleVerse 🧩⚡

PuzzleVerse is an interactive 8-Puzzle game with AI solvers and live algorithm benchmarking.

It allows users to:

- 🎮 Play the puzzle manually
- 🤖 Watch AI solve it (BFS, DFS, A*)
- 🏁 Run a live race between algorithms
- 📊 Compare performance metrics visually

---

## 🌐 Live Demo

👉 https://puzzleverse-indol.vercel.app

---

## ✨ Features

- Manual Play Mode
- AI Solve Mode
- Real-time Algorithm Race
- Performance Analytics Dashboard
- Neon Tech UI Design
- Fully Responsive Layout

---

## 🧠 Algorithms Implemented

- Breadth First Search (BFS)
- Depth First Search (DFS)
- A* Search Algorithm

Metrics compared:
- Nodes Expanded
- Solution Depth
- Execution Time
- Efficiency Score
